GroundHog Ratings.com (LAMP Stack)

A classic server-side movie rating web application built using the LAMP stack (Linux/Apache/MySQL/PHP) for the same IT course. This project introduces a unique rating concept: movies are rated based on the user's desired rewatchability interval (e.g., "I'd want to see this again in 5 years"). This demonstrates foundational knowledge of traditional server-side architecture, SQL database management, and PHP development.

Technologies Used:

  • Apache Web Server
  • MySQL (Database)
  • PHP (Backend Logic)

Virtual Reading Room

Building on a web scraping assignment from my 'Practical Programming in Python' course, I developed a full-stack web application. It lets users explore and read public domain books from Project Gutenberg, demonstrating responsible web scraping (using official catalog data), database integration, API development, and a dynamic frontend. Users can search, filter by language, browse, and view books directly in the browser.

Technologies Used:

  • Python (Flask for Backend)
  • Pandas (for data processing)
  • SQLite (for Database)
  • HTML, CSS, JavaScript (Frontend)
  • GitHub (for Version Control)
  • Render.com (for Deployment)
